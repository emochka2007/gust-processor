Processor emulator based on **Introduction to Machine and Assembly Language Programming by PHILIP J. GUST**


```bash
- cd rust-assembler
- cargo run ./src/factorial.asm  ../emulator.obj && cd .. && make && ./emulator && cd rust-assembler
```
